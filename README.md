# Umpei Miyamoto Website

Static GitHub Pages site for Umpei Miyamoto.

## Contents

- `index.html`: Japanese home page
- `en/index.html`: English home page
- `research.html`, `en/research.html`: research overview with MathJax
- `publications.html`, `en/publications.html`: arXiv-derived publication list
- `career.html`, `en/career.html`: career timeline and current positions
- `education.html`, `en/education.html`: teaching, textbook, and academic training
- `access.html`, `en/access.html`: access information and map for Akita Prefectural University Honjo Campus
- `album.html`, `en/album.html`: photo album using `assets/album/images`, `assets/album/field`, and `assets/album/featured`
- `assets/js/data.js`: generated publication and album data

## Deployment

Copy the contents of this directory to the root of `UmpeiMiyamoto.github.io` and push to GitHub Pages. No build step is required.

The only external runtime dependency is MathJax on the research and education pages.
